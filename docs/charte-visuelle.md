## Mises en évidence

- **Surlignage jaune** : fond #FFFF78 derrière les mots-clés importants dans les titres
- **Texte violet** : pour les titres de sections
- **Texte corail** : automatique sur les h3
- **Majuscules** : titres de sections et navigation
- Pas de gras (la police n'a qu'un seul poids : 500)
- Pas de souligné

## Responsive

Deux breakpoints :
- Tablette : 960px
- Mobile : 600px

Le site est conçu mobile-friendly. Les grilles passent en colonne unique sur mobile.
